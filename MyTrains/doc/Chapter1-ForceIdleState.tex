\subsection{ForceIdleState}

\rule{15.1cm}{0.4pt}

\underline{Description:}

This command forces the network into the idle state and broadcasts an emergency stop.

\underline{Group:}

\underline{Opcode:}

OPC\_IDLE

\underline{Type:}
