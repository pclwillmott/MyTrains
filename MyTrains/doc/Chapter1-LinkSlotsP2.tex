\subsection{LinkSlotsP2}\index{LinkSlotsP2}\index{Consisting}

\underline{Description:}

This \gls{Command} requests the command station to link slot SL1 to slot SL2. If the \gls{Command} was successful then a \textbf{LocoSlotDataP2} \gls{Response} will be returned. An invalid link will return a \textbf{InvalidLinkP2} \gls{Response}.

\underline{Protocol:}

2

\underline{Group:}

6-Byte Message

\underline{Opcode:}

OPC\_D4\_GROUP (unofficial mnemonic)

\underline{Type:}

\gls{Command}
