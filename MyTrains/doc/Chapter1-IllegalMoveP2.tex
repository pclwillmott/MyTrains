\subsection{IllegalMoveP2}\index{IllegalMoveP2}

\rule{15.1cm}{0.4pt}

\underline{Description:}

The \textbf{IllegalMoveP2} response means that the slot move request was not successful. This could be because the slot combination is invalid or that the slot in question was \gls{In-Use}.

\underline{Protocol:}

2

\underline{Group:}

4-Byte Message

\underline{Opcode:}

OPC\_LONG\_ACK

\underline{Type:}

Response
