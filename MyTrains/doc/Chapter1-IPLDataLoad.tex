\subsection{IPLDataLoad}\index{IPLDataLoad}

\rule{15.1cm}{0.4pt}

\underline{Description:}

This \gls{Command} loads firmware data into a device that supports IPL. D1 is the lowest addressed byte and D8 is the highest addressed byte.

\underline{Group:}

Variable-Byte Message

\underline{Opcode:}

OPC\_PEER\_XFER

\underline{Type:}

\gls{Command}
