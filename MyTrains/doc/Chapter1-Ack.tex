\subsection{Ack}\index{Ack}

\underline{Description:}

This message provides a response code from a command. This is the generic form of this message type.

\underline{Group:}

4-Byte Message

\underline{Opcode:}

OPC\_LONG\_ACK

\underline{Type:}
