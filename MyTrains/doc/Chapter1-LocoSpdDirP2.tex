\subsection{LocoSpdDirP2}\index{LocoSpdDirP2}\index{Speed}\index{Direction}

\underline{Description:}

This function sets the locomotive's speed in the range 0 to 127 and direction. 0 means inertial stop and 1 means emergency stop. Other values mean increasing speed.

\underline{Protocol:}

2

\underline{Group:}

6-Byte Message

\underline{Opcode:}

OPC\_D5\_GROUP (unofficial mnemonic)

\underline{Type:}

Command
