\subsection{Reset}
\underline{Description:}

This broadcast message is sent by a command station when its ``Loco Reset" button has been pressed. Software should reload any locally cached slot data from the command station.

\underline{Group:}

2-Byte Message

\underline{Opcode:}

OPC\_LOCO\_RESET

\underline{Type:}

Broadcast
