
OST1 to OST14 encode the command station's option switch table. A bit value of 1 means that the switch is closed and a value of 0 means that a switch is thrown. OpSw 8, OpSw 16, OpSw 24, OpSw 32, OpSw 40, OpSw 48, OpSw 56, OpSw 64, OpSw 72, OpSw 80, OpSw 88, OpSw 96, OpSw104, and OpSw 112 cannot be read due to bit 7 being cleared in the message format.
