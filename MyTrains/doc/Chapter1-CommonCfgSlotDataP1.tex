
OST1 to OST6 encode the command station's option switch table. A bit value of 1 means that the switch is closed and a value of 0 means that a switch is thrown. OpSw 8, OpSw 16, OpSw 24, OpSw 32 and OpSw 40 cannot be read due to bit 7 being cleared in the message format.

\begin{tabular}{p{0.05\linewidth} p{0.9\linewidth}}

d3 & 1 means the programming track is busy.\\
d2 & 1 means this master implements protocol 1 commands,  0 means the command station is a DT200.\\
