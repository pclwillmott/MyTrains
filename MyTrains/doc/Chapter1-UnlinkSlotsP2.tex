\section{UnlinkSlotsP2}\index{UnlinkSlotsP2}\index{Consisting}

\underline{Description:}

This \gls{Command} unlinks sub-member or mid-consist slot SL1 from top-member slot SL2. If the command was successful then a \textbf{LocoSlotDataP2} response will be returned for the sub-member or mid-consist slot. An invalid link will return a \textbf{Ack} with a response code of 0x00.

\underline{Protocol:}

2

\underline{Group:}

6-Byte Message

\underline{Opcode:}

OPC\_D4\_GROUP (unofficial mnemonic)

\underline{Type:}

\gls{Command}
