\subsection{UnlinkSlotsP2}

\underline{Description:}

This command unlinks slot SL1 from a consist. The command station sets SL\_CONUP/DN flags appropriately. If the command was successful then a \textbf{LocoSlotDataP2} response will be returned. An invalid link will return a \textbf{Ack} with a response code of 0x00.

\underline{Protocol:}

2

\underline{Group:}

6-Byte Message

\underline{Opcode:}

OPC\_D4\_GROUP (Unofficial mnemonic)

\underline{Type:}

Command
