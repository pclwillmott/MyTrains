\section{LinkSlotsP1}\index{LinkSlotsP1}\index{Consisting}

\underline{Description:}

This \gls{Command} requests the command station to link slot SL1 to slot SL2. If the \gls{Command} was successful then a \textbf{LocoSlotDataP1} \gls{Response} will be returned. An invalid link will return a \textbf{InvalidLinkP1} \gls{Response}.

\underline{Protocol:}

1

\underline{Group:}

4-Byte Message

\underline{Opcode:}

\underline{Type:}
